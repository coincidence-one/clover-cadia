## 🎮 게임 소개
**CloverCadia**는 **클래식 슬롯**과 **로그라이트 메카닉**이 결합된 독특한 게임입니다.  
단순한 도박이 아닙니다. **데드라인(기한)**과의 싸움입니다.  
스핀이 다 떨어지기 전에 목표 금액을 달성하여 다음 라운드로 진출하고, 티켓을 모아 강력한 아이템을 구매하세요.

## ✨ 주요 기능 (v2.0)

### 1. 🎟️ 티켓 시스템
- **티켓**은 라운드 클리어 보상으로 얻는 특수 화폐입니다.
- **티켓 상점**에서 영구적인 패시브 아이템이나 강력한 일회성 아이템을 구매할 수 있습니다.

### 2. ⏰ 데드라인 라운드 시스템
- 갈수록 어려워지는 라운드 목표를 달성해야 합니다.
- **1라운드:** 15 스핀 내에 300 코인 달성
- **2라운드:** 20 스핀 내에 800 코인 달성
- 스핀이 0이 되면... **게임 오버(GAME OVER)**.

### 3. 📞 폰 보너스 시스템 (Phone Bonus)
- 라운드를 클리어하면 **의문의 전화**가 걸려옵니다.
- 3가지 보너스 중 하나를 선택하세요:
  - 🎁 **버프:** 코인/티켓 지급 또는 영구 확률 증가.
  - 😈 **리스크:** 하이 리스크 하이 리턴 (예: 코인 절반을 바치고 스핀 +10).

### 4. 🕹️ CloverPit 스타일 경제
- **고정 스핀 비용:** 10 코인.
- **베팅 전략 없음:** 순수 운과 아이템 관리 능력으로 승부합니다.
- **심볼 가치 기반:** `심볼 가치 x 매치 배수`로 당첨금이 계산됩니다.

## 🛠️ 기술 스택
- **프레임워크:** Next.js 16 + React 19
- **언어:** TypeScript
- **스타일링:** Tailwind CSS v4 (Pixel Art 스타일)
- **UI 컴포넌트:** Radix UI, Lucide React
- **백엔드:** Supabase
- **상태 관리:** Custom React Hooks
- **오디오:** Web Audio API (Chiptune 효과음)
- **유틸리티:** Sonner (토스트), clsx, tailwind-merge

## 🚀 실행 방법

```bash
# 의존성 설치

```
