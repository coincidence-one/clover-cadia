## ✨ 주요 기능

### 🎲 5×3 슬롯 그리드
- 15칸 그리드, 7개 페이라인 (가로 + 대각선)
- 밸런스 있는 확률 기반 심볼 시스템

### 🍀 심볼 & 확률
| 심볼 | 이름 | 확률 | 배당 |
|------|------|------|------|
| 🍒🍋 | 체리 / 레몬 | 19.4% | 2배 |
| ☘️🔔 | 클로버 / 종 | 14.9% | 5배 |
| 💎💰 | 다이아 / 보물 | 11.9% | 10배 |
| 7️⃣ | 럭키 세븐 | 7.5% | 25배 |
| 6️⃣ | 저주 | 1.5% | ☠️ |

### ☠️ 666 저주
> *6이 세 개 나오면... 모든 것을 잃는다.*

666이 뜨면 보유 코인이 전부 사라집니다! 하지만 **성스러운 방패** 아이템으로 막을 수 있어요.

### 🛒 아이템 상점
| 아이템 | 효과 |
|--------|------|
| 🍀 럭키 참 | 클로버 확률 증가 (3회) |
| ⭐ 더블 스타 | 다음 당첨 시 2배 |
| 🔥 핫 스트릭 | 보너스 스핀 +5 |
| ✝️ 성스러운 방패 | 666 저주 방어 |
| 🃏 와일드 카드 | 와일드 심볼 추가 |

### 🏆 업적 시스템
8개의 업적을 달성하세요:
- 🏆 첫 당첨
- 7️⃣ 럭키 세븐 (7 x 3 달성)
- 💎 잭팟 헌터 (7 x 5 달성)
- 😈 저주받은 자 (666 발동)
- ✝️ 생존자 (방패로 666 방어)

### 🌍 한국어 / 영어 지원
버튼 하나로 언어 전환!

## 🛠️ 기술 스택

- **프레임워크**: Next.js 16 (App Router)
- **UI**: 8bitcn/ui + Tailwind CSS
- **오디오**: Web Audio API (칩튠 엔진)
- **폰트**: Press Start 2P
- **저장**: localStorage
- **배포**: Vercel

## 🚀 로컬 실행

```bash
# 클론
git clone https://github.com/coincidence-one/clover-cadia.git
cd clover-cadia

# 설치
npm install

# 실행
npm run dev
```

[http://localhost:3000](http://localhost:3000) 에서 스핀!
